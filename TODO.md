# kids-quiz-app
## このプロジェクトについて

このプロジェクトは、「小学5年生は賢いの？」というテレビ番組に基づいたクイズを作成し、そのクイズアプリを作成します。 
使う技術はHTML、CSS、JavaScriptを用います。 
クイズの内容は、国語、算数、社会、理科、英語などのクイズを作成します。また、このクイズはJSONファイルに格納し、国語のJSONファイル、算数のJSONファイルのように、各科目ごとにJSONファイルを分けて作成します。 
HTML, CSS, JavaScriptを用いて、クイズの正解や現在の問題数などはローカルストレージのJavaScriptの関数を用い、そこからJSONファイルからランダムで問題を持ってきてやります。
クイズのモードはランダムで10問出題するモードと、科目ごとに分けて出題するモードの2パターンを用意します。

画面上には、クイズに正解するごとにお金が増えていくというシステムを採用し、またクイズ画面では上に問題と下に選択肢を4つ用意し、ライフを表示したり、あとはタイムのバーを表示したりします。各問題は30秒程度にする予定です。 
index.htmlにはそのモードを選択できるようにしました。一度やった問題はおさらいできるようにしたりします。 また、クイズの表示する場所には quiz.html を作成し、JSONファイルから各問題を持ってこられるようにし、ランダムで出します.
